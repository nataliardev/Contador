# 🔢 Contador

Aplicação web simples de **contador** desenvolvida com **HTML**, **CSS** e **JavaScript**, com persistência de dados utilizando **localStorage**.

---

## 📌 Sobre o projeto

O **Contador** é um projeto front-end básico criado para praticar conceitos fundamentais de desenvolvimento web.

A aplicação permite ao usuário:

- ➕ Incrementar o valor do contador
- ➖ Decrementar o valor do contador
- 🔄 Reiniciar o contador
- 💾 Manter o valor salvo mesmo após atualizar ou fechar a página

O valor do contador é armazenado no navegador utilizando **localStorage**, garantindo que o número permaneça salvo entre sessões.

---

## 🛠 Tecnologias utilizadas

Este projeto foi desenvolvido utilizando:

- **HTML5**
- **CSS3**
- **JavaScript**
- **LocalStorage (Web API)**

---

## ⚙️ Funcionalidades

- Exibição dinâmica do valor do contador
- Botão para aumentar o valor
- Botão para diminuir o valor
- Botão para reiniciar o contador
- Salvamento automático do valor no navegador
